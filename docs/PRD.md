# PRD — Bridgemakers Website

## 목표

브릿지메이커스 크리에이티브 에이전시의 포트폴리오 웹사이트 및 관리자 대시보드를 운영한다. 잠재 클라이언트에게 작업물을 소개하고, 내부 관리자가 프로젝트·문의를 직접 관리할 수 있는 시스템을 제공한다.

## 사용자

- **잠재 클라이언트**: 공개 페이지에서 포트폴리오 탐색, 문의 제출
- **관리자(Bridgemakers 직원)**: 대시보드에서 프로젝트 CRUD, 문의 관리, 사이트 콘텐츠 업데이트

## 핵심 기능

1. **포트폴리오 공개 페이지**: 프로젝트 목록·상세 페이지, 카테고리·태그 필터링, 관련 프로젝트 추천
2. **다국어 지원**: 한국어(ko), 영어(en), 독일어(de) 전환 가능. URL 구조: `/{lang}/...`
3. **관리자 대시보드**: 프로젝트 생성·수정·삭제, 이미지 업로드(Supabase Storage), 공개/비공개 설정, 문의 내역 조회
4. **인증 시스템**: Supabase Auth 기반 이메일+OTP 로그인, Google OAuth, 역할 기반 접근 제어
5. **문의하기(Contact Us)**: 이메일 폼 제출, Supabase에 저장, 관리자 알림

## MVP 제외 사항

- 블로그/뉴스 섹션 (추후 추가)
- 다국어 자동 번역 (수동 번역 파일로 운영)
- 결제/견적 기능

## 디자인

- **철학**: Apple Human Interface Guidelines 기반, 직관적이고 일관된 사용자 경험
- **컬러**: 배경 `#000000`, 강조 금색 `#cba967`, 텍스트 `#FFFFFF` / 그레이 계열
- **폰트**: SF Pro Display, Noto Sans KR, Apple SD Gothic Neo, Inter
- **레이아웃**: 최대 너비 1440px, 12칸 그리드, 섹션 상하 여백 80px

## 현재 상태 (2026-04-21 기준)

- 공개 페이지: 홈, About, 포트폴리오, Contact 구현됨
- 대시보드: 프로젝트 관리 페이지 구현됨 (정렬·페이지네이션 포함)
- 인증: Supabase Auth (이메일+OTP, Google OAuth) 구현됨
- 다국어: ko/en/de 번역 파일 구조 완성, 영문 번역 일부 미완성
